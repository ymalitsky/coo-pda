rb-random_svd-1

This documents contains detailed results for the robust PCA problem $$\min_{L,S} \|L\|_* + \lambda \|S\|_1\qquad \text{s.t.}\quad L+S = M,$$ where \(L, S, M\in \mathbb{R}^{n_1\times n_2}\), \(\lambda > 0\).

For given \(n_1,n_2\) and rank \(r\) we set \(L^\dagger = Q_1 Q_2\) with \(Q_1 \in \mathbb{R}^{n_1 \times r}\) and \(Q_2 \in \mathbb{R}^{r\times n_2}\), whose elements are i.i.d. \(\mathcal N (0,1)\) random variables. Then we generate \(S^\dagger \in \mathbb{R}^{n_1\times n_2}\) as a sparse matrix whose \(5\%\) non-zero elements are chosen uniformly at random from the range \([-500, 500]\). We set \(M = L^\dagger + S^\dagger\) and \(\lambda = \frac{1}{\sqrt{n_1}}\).

Here we use the randomized evaluation of the SVD from the sklearn module. The residual measures ||Lk - L† || — the distance to the true matrix \(L^\dagger\).
